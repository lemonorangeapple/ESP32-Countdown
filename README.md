# ESP32-Countdown

项目结构
```plaintext
--- countdown-com 串口通信版本
--- countdown-u8g2 带字库显示屏版本
```

接线：
```plaintext
[ ESP32-C3 Super Mini ]                      [ 7-Pin OLED Display ]
     +-----------------+                           +----------------+
     |             3V3 |-------------------------->| VCC            |
     |             GND |-------------------------->| GND            |
     |           GPIO1 |-------------------------->| SDA (MOSI)     |
     |           GPIO0 |-------------------------->| CS             |
     |           GPIO4 |-------------------------->| SCL (SCK)      |
     |           GPIO2 |-------------------------->| DC             |
     |           GPIO3 |-------------------------->| RST            |
     +-----------------+                           +----------------+
```
